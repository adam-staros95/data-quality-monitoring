# Getting started

Include dependency:

```scala
"com.github.piotr-kalanski" % "data-quality-monitoring_2.11" % "0.1.0"
```

or

```xml
<dependency>
    <groupId>com.github.piotr-kalanski</groupId>
    <artifactId>data-quality-monitoring_2.11</artifactId>
    <version>0.1.0</version>
</dependency>
```
